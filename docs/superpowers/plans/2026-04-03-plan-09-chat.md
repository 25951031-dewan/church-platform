# Plan 9: Real-Time Chat — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a real-time chat system with 1-on-1 and group messaging, typing indicators, read receipts, media sharing, presence tracking, and chat moderation. Use Laravel Broadcasting abstraction with Pusher driver (shared hosting) or Reverb (VPS self-hosted) — same codebase works with both.

**Architecture:** Laravel Broadcasting with private channels for conversations. Pusher driver for shared hosting deployment (zero server setup), Reverb driver as VPS alternative (self-hosted, no monthly fees). Frontend uses Laravel Echo + Pusher.js. Messages stored in MySQL, real-time delivery via WebSocket. Presence channels for online/offline status.

**Tech Stack:** Laravel 12 Broadcasting, Pusher (production) / Reverb (VPS), Laravel Echo, Pusher.js, React 19, TypeScript 5.8, TanStack React Query 5, Zustand 5, Redis 7 (cache + presence)

**Spec:** `docs/superpowers/specs/2026-03-28-church-community-platform-design.md` (Week 9-10: Real-Time Chat, Chat Plugin section)

**Broadcasting Strategy:** Write once, deploy anywhere. Laravel Broadcasting abstraction means identical backend/frontend code works with Pusher (shared hosting, better docs, free tier) OR Reverb (VPS, self-hosted, no fees). Just change `BROADCAST_DRIVER` in `.env`.

## Database Schema

### conversations

| Column | Type | Description |
|--------|------|-------------|
| id | bigint PK | Auto-increment |
| type | enum('direct', 'group') | 1-on-1 or group chat |
| name | varchar(255) nullable | Group chat name (null for direct) |
| created_by | bigint FK → users.id | Creator user ID |
| created_at | timestamp | |
| updated_at | timestamp | Last message time (for sorting) |

**Indexes:** created_by, type, updated_at DESC

### messages

| Column | Type | Description |
|--------|------|-------------|
| id | bigint PK | Auto-increment |
| conversation_id | bigint FK → conversations.id | Parent conversation |
| user_id | bigint FK → users.id | Sender |
| body | text nullable | Message text (null for media-only) |
| type | enum('text', 'image', 'file', 'audio') | Message type |
| file_entry_id | bigint nullable FK → file_entries.id | Attached media |
| created_at | timestamp | Sent at |
| deleted_at | timestamp nullable | Soft delete |

**Indexes:** conversation_id + created_at DESC, user_id

### conversation_user (pivot)

| Column | Type | Description |
|--------|------|-------------|
| id | bigint PK | Auto-increment |
| conversation_id | bigint FK → conversations.id | |
| user_id | bigint FK → users.id | Participant |
| last_read_at | timestamp nullable | Last read message timestamp |
| is_muted | boolean default false | Mute notifications |
| joined_at | timestamp | When user joined (for groups) |

**Indexes:** conversation_id + user_id (unique), user_id

```php
namespace Common\Chat\Models;

use Illuminate\Database\Eloquent\Model;
use Illuminate\Database\Eloquent\Relations\BelongsTo;
use Illuminate\Database\Eloquent\Relations\BelongsToMany;
use Illuminate\Database\Eloquent\Relations\HasMany;
use Common\Auth\Models\User;

class Conversation extends Model
{
    protected $fillable = ['type', 'name', 'created_by'];
    
    protected $casts = [
        'created_at' => 'datetime',
        'updated_at' => 'datetime',
    ];
    
    public function users(): BelongsToMany
    {
        return $this->belongsToMany(User::class, 'conversation_user')
            ->withPivot(['last_read_at', 'is_muted', 'joined_at'])
            ->withTimestamps();
    }
    
    public function messages(): HasMany
    {
        return $this->hasMany(Message::class)->orderBy('created_at', 'desc');
    }
    
    public function latestMessage(): HasOne
    {
        return $this->hasOne(Message::class)->latestOfMany();
    }
    
    public function creator(): BelongsTo
    {
        return $this->belongsTo(User::class, 'created_by');
    }
    
    // Helper: Get unread count for a user
    public function unreadCountFor(User $user): int
    {
        $lastRead = $this->users()
            ->where('user_id', $user->id)
            ->first()
            ?->pivot
            ?->last_read_at;
        
        return $this->messages()
            ->where('user_id', '!=', $user->id)
            ->when($lastRead, fn($q) => $q->where('created_at', '>', $lastRead))
            ->count();
    }
    
    // Helper: Is user a participant?
    public function hasUser(User $user): bool
    {
        return $this->users()->where('user_id', $user->id)->exists();
    }
}
```

    {
        return $conversation->hasUser($user) || $user->hasPermission('chat.moderate');
    }
    
    public function create(User $user): bool
    {
        return $user->hasPermission('chat.send');
    }
    
    public function createGroup(User $user): bool
    {
        return $user->hasPermission('chat.create_group');
    }
    
    public function send(User $user, Conversation $conversation): bool
    {
        return $conversation->hasUser($user) && $user->hasPermission('chat.send');
    }
    
    public function attachFiles(User $user): bool
    {
        return $user->hasPermission('chat.attach_files');
    }
    
    public function moderate(User $user): bool
    {
        return $user->hasPermission('chat.moderate');
    }
    
    public function delete(User $user, Conversation $conversation): bool
    {
        return $conversation->created_by === $user->id || $user->hasPermission('chat.moderate');
    }
}

```php
namespace Common\Chat\Controllers;

use Common\Chat\Models\Conversation;
use Common\Chat\Requests\CreateConversationRequest;
use Illuminate\Http\JsonResponse;
use Illuminate\Http\Request;
use App\Http\Controllers\Controller;

class ConversationController extends Controller
{
    // GET /api/v1/chat/conversations
    public function index(Request $request): JsonResponse
    {
        $conversations = $request->user()
            ->conversations()
            ->with(['latestMessage.user', 'users'])
            ->get()
            ->map(fn($conv) => [
                'id' => $conv->id,
                'type' => $conv->type,
                'name' => $conv->name,
                'users' => $conv->users,
                'latest_message' => $conv->latestMessage,
                'unread_count' => $conv->unreadCountFor($request->user()),
                'updated_at' => $conv->updated_at,
            ]);
        
        return response()->json($conversations);
    }
    
    // GET /api/v1/chat/conversations/{conversation}
    public function show(Conversation $conversation): JsonResponse
    {
        $this->authorize('view', $conversation);
        
        return response()->json($conversation->load('users'));
    }
    
    // POST /api/v1/chat/conversations
    public function store(CreateConversationRequest $request): JsonResponse
    {
        $this->authorize('create', Conversation::class);
        
        $type = count($request->user_ids) > 1 ? 'group' : 'direct';
        
        if ($type === 'group') {
            $this->authorize('createGroup', Conversation::class);
        }
        
        // Check if direct conversation already exists
        if ($type === 'direct') {
            $existing = Conversation::where('type', 'direct')
                ->whereHas('users', fn($q) => $q->where('user_id', $request->user()->id))
                ->whereHas('users', fn($q) => $q->where('user_id', $request->user_ids[0]))
                ->first();
            
            if ($existing) {
                return response()->json($existing->load('users'));
            }
        }
        
        $conversation = Conversation::create([
            'type' => $type,
            'name' => $request->name,
            'created_by' => $request->user()->id,
        ]);
        
        // Attach participants (include creator)
        $participants = array_unique([...$request->user_ids, $request->user()->id]);
        $conversation->users()->attach($participants);
        
        return response()->json($conversation->load('users'), 201);
    }
    
    // POST /api/v1/chat/conversations/{conversation}/read
    public function markAsRead(Conversation $conversation, Request $request): JsonResponse
    {
        $this->authorize('view', $conversation);
        
        $conversation->users()
            ->updateExistingPivot($request->user()->id, [
                'last_read_at' => now(),
            ]);
        
        broadcast(new \Common\Chat\Events\MessageRead(
            $conversation->id,
            $request->user()->id,
            now()->toISOString()
        ));
        
        return response()->json(['message' => 'Marked as read']);
    }
    
    // DELETE /api/v1/chat/conversations/{conversation}
    public function destroy(Conversation $conversation, Request $request): JsonResponse
    {
        $this->authorize('delete', $conversation);
        
        // Remove user from conversation (leave)
        $conversation->users()->detach($request->user()->id);
        
        return response()->json(['message' => 'Left conversation']);
    }
}
```

        $this->authorize('view', $conversation);
        
        $messages = $conversation->messages()
            ->with(['user', 'fileEntry'])
            ->orderBy('created_at', 'desc')
            ->paginate(50);
        
        return response()->json($messages);
    }
    
    // POST /api/v1/chat/conversations/{conversation}/messages
    public function store(Conversation $conversation, SendMessageRequest $request): JsonResponse
    {
        $this->authorize('send', $conversation);
        
        if ($request->file_entry_id && !auth()->user()->can('attachFiles', Conversation::class)) {
            abort(403, 'You do not have permission to attach files');
        }
        
        $message = $conversation->messages()->create([
            'user_id' => $request->user()->id,
            'body' => $request->body,
            'type' => $request->type ?? 'text',
            'file_entry_id' => $request->file_entry_id,
        ]);
        
        // Update conversation timestamp
        $conversation->touch();
        
        // Broadcast message
        broadcast(new MessageSent($message->load('user', 'fileEntry')));
        
        return response()->json($message->load('user', 'fileEntry'), 201);
    }
    
    // DELETE /api/v1/chat/messages/{message}
    public function destroy(Message $message, Request $request): JsonResponse
    {
        if ($message->user_id !== $request->user()->id && !$request->user()->hasPermission('chat.moderate')) {
            abort(403, 'Unauthorized');
        }
        
        $message->delete(); // Soft delete
        
        return response()->json(['message' => 'Message deleted']);
    }
}

```typescript
import { useQuery, useQueryClient } from '@tanstack/react-query';
import { apiClient } from '@common/http/query-client';
import { useEffect } from 'react';
import echo from '@/lib/echo';

export interface Message {
  id: number;
  conversation_id: number;
  user_id: number;
  body: string | null;
  type: 'text' | 'image' | 'file' | 'audio';
  file_entry_id: number | null;
  user: User;
  fileEntry?: FileEntry;
  created_at: string;
}

export function useMessages(conversationId: number) {
  const queryClient = useQueryClient();
  
  const query = useQuery({
    queryKey: ['messages', conversationId],
    queryFn: () =>
      apiClient.get<{ data: Message[] }>(`/chat/conversations/${conversationId}/messages`)
        .then(r => r.data.data),
    enabled: !!conversationId,
  });
  
  // Subscribe to new messages
  useEffect(() => {
    if (!conversationId) return;
    
    const channel = echo.private(`conversation.${conversationId}`);
    
    channel.listen('MessageSent', (e: { message: Message }) => {
      queryClient.setQueryData<Message[]>(['messages', conversationId], (old = []) => {
        // Avoid duplicates
        if (old.some(m => m.id === e.message.id)) return old;
        return [...old, e.message];
      });
    });
    
    return () => {
      channel.stopListening('MessageSent');
      echo.leave(`conversation.${conversationId}`);
    };
  }, [conversationId, queryClient]);
  
  return query;
}
```

import { useEffect, useState } from 'react';
import echo from '@/lib/echo';

export function usePresence() {
  const [onlineUsers, setOnlineUsers] = useState<number[]>([]);
  
  useEffect(() => {
    const channel = echo.join('online');
    
    channel
      .here((users: { id: number }[]) => {
        setOnlineUsers(users.map(u => u.id));
      })
      .joining((user: { id: number }) => {
        setOnlineUsers(prev => [...prev, user.id]);
      })
      .leaving((user: { id: number }) => {
        setOnlineUsers(prev => prev.filter(id => id !== user.id));
      });
    
    return () => {
      echo.leave('online');
    };
  }, []);
  
  return { onlineUsers, isOnline: (userId: number) => onlineUsers.includes(userId) };
}

## Notes

**Pusher vs Reverb Decision Matrix:**

| Requirement | Recommended Driver |
|-------------|-------------------|
| Shared hosting (cPanel, Bluehost) | Pusher |
| VPS with SSH access | Reverb |
| Low traffic (<100 concurrent users) | Pusher (free tier) |
| High traffic (>100 users) | Reverb (cost-effective) |
| Quick deployment | Pusher (zero config) |
| Full control | Reverb (self-hosted) |

**Both drivers work with identical code** — just change `BROADCAST_DRIVER` in `.env`. This flexibility is a key advantage of Laravel Broadcasting abstraction.
